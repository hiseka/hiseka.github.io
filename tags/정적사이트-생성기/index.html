 # 블로그를 옮기는 것에 실패하다.

점점 블로그에 접속자체를 안하게 되고 관심도 시들어지면서 2019년에는 새롭게 블로그를 다시 꾸며보고 싶었다. 그렇타고 이전의 블로그를 날리는 것이 아니라 새로움만을 느끼면서 계속 유지하고 싶었다. 그래서 생각한 것이 블로그의 디자인 변경이였다. 그렇게 디자인을 변경하기 위해 여러 사이트를 돌아다니면서 헥소의 태마를 살펴보았지만 더 나은 태마를 찾지 못했다

팔로우:

공지사항

  1. 기존 Living - Interest > Living - Money 카테고리로 재분리
  2. 기존 Living - Game > Living - Interest 카테고리로 통합

  5. 태그 클라우드를 사이드 바에서 페이지로 신설
