해외 결제 체크카드 정리(2019년 03월 기준)

 # 신한 스마트 글로벌 체크카드 단종 기념
2019-03-20
Check Card, Fees, Money, Overseas Payment, Saving, 돈, 수수료, 저축, 체크카드, 해외 결제

해외 결제 체크카드 중에서 원톱이였던 신한 스마트 글로벌 체크카드가 단종되면서 해외 결제 체크카드에 지각 변동이 있었습니다.

그래서 다시 정리해 봅니다.

PS : 해당 글은 뽐뿌에 제가 올렸던 글을 백업을 목적으로 옮겨온 것입니다.

보는 법

카드 이름

발행 카드사 / 연회비 유무 / 전월 실적 유무 / 조건 유무(만 18세 이상 발급 가능, 미성년자는 보호자 동의 필요등 당연한 조건에 대해서는 언급하지 않는다.) / 연결은행

해택 설명

<비자/VISA>(해외 결제 수수료 1.1%. 하지만, 현제 국내 카드사와 소송 중이며 상승분 0.1%는 국내 카드사에서 부담하고 있어 실부담 수수료는 1%이다.)

VIVA+ 플레티늄 체크카드

하나카드 / 무연회비 / 무실적 / 무조건 / 하나은행

비자 1%, 하나카드 건당 0.5$이 모두 면제된다. 실 수수료 0원이 청구되는 카드이다.(기존 네이버 페이 체크카드를 눌러버린 비자 브랜드의 강자이다.)

페이코 우리 체크카드

우리카드(비씨카드) / 무연회비 / 무실적 / 무조건 / 페이코 연결 계좌 가능 은행 중 일부(우리, 국민, 기업, 농협, 신한, 하나, 경남, 부산, 대구, 광주, 추후 늘어날 가능성이 있습니다.)

비자 1%, 우리카드 건당 0.5$이 청구되지만 사용액의 1.5% 포인트 적립이기 때문에 100$(정확히 비자 수수료가 1$ 그리고 우리카드 수수료 0.5$을 상쇄할 수 있는 금액) 이상 결제시 수수료 측면에서 유리하다. 참고로 포인트 적립은 1달에 최대 1만 포인트까지 쌓을 수 있다. 2018년 12월까지는 1만 5천 포인트까지 적립하는 이벤트를 진행 중이다. 페이코 앱에서만 발급 신청이 가능하다.

<마스터/MASTER>(해외 결제 수수료 1%이다.)

Fantastic S 체크카드

신한카드 / 무연회비 / 무실적 / 무조건 / 신한은행

마스터 1%, 신한카드 0.2%가 청구되지만 1.2% 캐시백 되기 때문에 체감 수수료는 0원이 되는 카드이다.(잠깐! 수수료는 모두 곱셈으로 적용된다. 즉 마스터 1%와 신한카드 0.2%의 수수료는 정확히 1.2%가 아닌 그보다 약간 많은 요컨데 100$을 사용한다면 대략 1.2$ 아닌 1.202$이 청구된다. 따라서 엄밀히 말하자면 수수료가 캐시백으로 인해 아예 상쇄되지는 않는다.)

글로벌 언리미티드 체크카드

NH카드 / 3000원 / 무실적 / 무조건 / 농협은행

마스터 1%, NH카드 건당 0.5$이 청구되지만 사용 금액의 2%가 캐쉬백 된다. 연회비와 건당 수수료 때문에 소액 결제보다는 해외에서 고액 결제 할때 유리한 체크카드이다.

캐시백 체크카드

씨티카드 / 무연회비 / 무실적 / 하단에서 설명 / 씨티은행

마스터 1%, 씨티카드 0.25%의 수수료가 청구되지만 각 고객 세그먼트(등급)에 따라 캐쉬백 요율이 달라지는 카드이다.(물론 카드명도 약간 다르기는 하지만 파생상품이라 함께 다루겠다.)

  • 평잔 5천만원 미만 / 세그먼트 : 일반 / 1% 캐쉬백 / 체감 수수료 0.25%
  • 평잔 5천만원 이상 2억원 미만 / 세그먼트 : 씨티 프라이어리티 / 1.5% 캐쉬백 / 체감 수수료 -0.25%
  • 평잔 2억원 이상 10억원 미만 / 세그먼트 : 씨티골드 / 1.7% 캐쉬백 / 체감 수수료 -0.45%
  • 평잔 10억원 이상 / 세그먼트 : 씨티골드 프라이빗 / 2% 캐쉬백 / 체감 수수료 -0.75%
    꿈의 카드이다.
체인지업 체크카드

신한카드 / 무연회비 / 무실적 / 무조건 / 신한은행

마스터 1%, 신한카드 0.2%의 수수료가 모두 면제된다. 실 수수료 0원이 청구되는 카드이다.
신한은행 창구에서만 발급이 가능하며 발급 당시 체인지업 외화통장을 만들거나 외화통장을 연결계좌로 삼을 수 있다. 결제시 금액을 USD로 환산한 USD를 홀딩한 다음 빼가는 방식이다. 인터넷 뱅킹이나 앱을 통해 기존에 사용하던 신한은행 통장(원화)에서 돈을 달러로 바꾸어 입금이 가능하고 지정 금액 만큼 외화를 매수하거나 매도해서 환햇지를 할수도 있는 상품이다. 또한 실물 USD등을 바로 입금해서 이 카드를 통해 사용도 가능하다. 바로 아래서 언급할 무슨 카드보다는 괭장히 메리트가 있는 카드라고 할 수 있다.
참고로 원화를 달러로 환전시 우대 환울이 적용된다.(전신환 매도율) 우대환율까지 생각한다면 수수료가 0% 체크카드보다 이득일수도 있다.

카드의정석 외화바로 체크카드

우리카드(비씨카드) / 무연회비 / 무실적 / 무조건 / 우리은행

마스터 1%, 우리카드 건당 0.5$의 수수료가 모두 면제된다. 실 수수료 0원이 청구되는 카드이다.
나머지는 신한은행이 우리은행으로 바뀐 것만 빼면 위의 체인지업 체크카드와 동일하다.

<아메리칸 엑스프레스/AMEX>(해외 결제 수수료 1.4%이다.)

글로벌페이 체크카드(블루/그래이로 한 사람에 한해 2장을 가질 수 있는 카드이다.)

하나카드 / 무연회비 / 무실적 / 무조건 / 하나은행

아멕스 1.4%, 하나카드 건당 0.5$가 청구되지만 결제 화폐가 USD(미합중국 달러)일 경우 앞의 수수료가 모두 면제되어 실 수수료 0원이 청구된다.
하나은행 창구에서만 발급이 가능하며 발급 당시 글로벌 페이 외화(USD 달러)통장, 글로벌 페이 원화 통장을 함께 개설해야 한다. 사용법은 글로벌 페이 원화 통장에 1만원 이상 돈을 넣게 되면 해당 금액에 맞추어 그날 고시된 전시환 매도율을 적용한 환율로 계산하여 USD로 전환해서 외화 통장으로 입금된다. 결제시 해당 외화 통장에 입금된 USD를 차감하여 결제가 가능하다. 이런 시스템 덕분에 환율이 쌀때 대량으로 원화를 USD로 바꾸어 차익을 볼 수 있으나 아주 애석하게도 외화 통장의 USD는 출금이 되지 않는다. 심지어 실물 USD도 입금이 되지 않는다. 약간 남은 USD도 원화로 바꾸어 이체도 되지 않는다.(창구에서 남은 USD를 환전 수수료를 내고 원화로 인출하거나 자신의 원화통장으로 이체 또는 외화 실물로 인출이 가능하다. 다만 실물 화폐로 환전되는 환율에 따라서 적용되어서 상당히 높은 수수료가 들어간다.)
선불카드로 보는 것이 타당할 정도로 까다로운 카드임에도 잘만 이용한다면 변팔 등 약간 불법에 가까운 껄쩍지근하면서도 이익을 볼 수 있는 해외 서비스를 이용하여 잘 결제되지 않는 곳에서 직구가 가능하다.(물론 이런 행동은 안하는 것이 좋타.)

<유니온 페이/CUP, Union Pay>(해외 결제 수수료 0.8%이다. 하지만 사용자가 부담하는 수수료는 없다.)

Deep Dream 체크카드(기본 디자인) / SOL Deep Dream 체크카드(기본 디자인)

신한카드 / 무연회비 / 무실적 / 무조건 / 신한은행(Deep Dream 체크카드의 경우 다른 은행도 연결이 가능하다.)

유니온 페이 0%, 신한은행 0.2%가 청구지만 결제 금액의 0.2%를 마이 신한포인트로 적립해 주기 때문에 체감 수수료는 0원이 되는 카드이다. 유니온 페이 자체가 결제망이 거의 없어 위의 카드들보다는 메리트가 떨어지지만 중국 현지 결제를 하거나 중국 쇼핑몰 온라인 신판(특히 샤오미 공홈)에서 먹히는 은련 체크카드이다.

<비씨 글로벌/BC Global>(해외 결제 수수료가 없다.)

뉴 현대 백화점 체크카드

우리카드(비씨카드) / 무연회비 / 무실적 / 무조건 / 우리은행(타 은행도 된다고 한다.)

비씨 글로벌 0%, 우리카드 건당 0.5$가 청구된다. 즉 건당 0.5$ 수수료가 발생되는 카드이다. 이 카드는 보통 직구족들이 아마존 등의 사이트에서 디스커버 카드 무료 배송 이벤트(비씨 글로벌이 디스커버와 제휴했기 때문에 미국 아마존 등에서는 디스커버 카드로 인식된다.)에 주로 활용하는 카드이다.
